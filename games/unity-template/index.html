Unity WebGL ビルドを読み込み中…
Unity WebGL ビルドが見つかりませんでした。
`Build/.loader.js` を `games/unity-template/Build/` に配置してください。

Unity の WebGL ビルドをこのフォルダに置くと、GameMarket 上で公開できます。

Unity の WebGL ビルドを公開するには、Unity のビルド設定で WebGL を選び、生成された `Build/` と `TemplateData/` をこのフォルダにコピーしてください。
